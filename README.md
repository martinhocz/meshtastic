Need to install meshtastic and pubsub python library
`pip3 install meshtastic pubsub`

On mac you need to create venv
```
python3 -m venv meshtastic
cd meshtastic
source ./bin/activate
pip3 install meshtastic
pip3 install pubsub
```
Or you can:
```
python3 -m venv meshtastic
cd meshtastic
source ./bin/activate
pip install -r requirements.txt
```
